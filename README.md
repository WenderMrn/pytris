# Pytris

A simple python game that runs on the console

## Dependencies

- python 3.13.5
- pysqlite 2.8.3
- blessed 1.21.0

### Create ENV

```bash
python -m venv env
```

### Active ENV

```bash
source env/bin/activate
```

### Deactivate

```bash
deactivate
```

### Create requiments.txt

pip freeze > requirements.txt

### Run on watch mode

watchfiles 'python main.py' .

### Keep console size

Keep the console at a minimum of 100x30
